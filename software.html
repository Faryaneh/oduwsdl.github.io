Software

The Dark and Stormy Archives project has started several software projects for community use. These projects help users analyze, reduce, and visualize web archive collections.

Archive-It Utilities (AIU)

  • Source code
  • Python Package Distribution Page

Archive-It Utilities is a Python library that gathers seed URLs and metadata from Archive-It collections via screen scraping and knowledge of Archive-It APIs.

The Off Topic Memento Toolkit (OTMT)

  • Source code
  • Python Package Distribution Page

The Off-Topic Memento Toolkit detects and identifies mementos that may be off-topic in a web archive collection. It does so through the use of topic modeling and similarity measures. This is a necessary step to discovering the best mementos in the collection for use in summarization and storytelling.

MementoEmbed

  • Source code
  • Documentation

MementoEmbed produces surrogates of individual mementos from web archives. These surrogates represent the mementos in forms such as cards or thumbnails for use in social media storytelling.

Raintale

  • Source code

Raintale takes the data generated from MementoEmbed and produces full stories of mementos suitable for publishing using services such as Twitter, Jekyll, or Blogger.

Hypercane (Coming Soon)

Hypercane will select the best mementos from a collection using a variety of different methods. It will use content provided by other components such as Archive-It utilities, the Off-Topic Memento Toolkit, and MementoEmbed.
