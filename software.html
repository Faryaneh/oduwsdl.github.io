Software

Archive-It Utilities

  • Source code
  • Python Package Distribution Page

Archive-It Utilities is a Python library that gathers seed URLs and metadata from Archive-It collections via screen scraping and knowledge of Archive-It APIs.

The Off Topic Memento Toolkit

  • Source code
  • Python Package Distribution Page

The Off-Topic Memento Toolkit detects and identifies mementos that may be off-topic in a web archive collection. It does so through the use of topic modeling and similarity measures.

MementoEmbed

  • Source code
  • Documentation

MementoEmbed produces surrogates of individual mementos from web archives. These surrogates represent the mementos in forms such as cards or thumbnails for use in social media storytelling.

Raintale (In Development)

  • Source code

Raintale takes the data generated from MementoEmbed and produces full stories of mementos suitable for publishing using Twitter, Jekyll, or Blogger.

Hypercane (Coming Soon)

Hypercane will select the best mementos from a collection using a variety of different methods.
